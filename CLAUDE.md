# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

A personal learning lab for the Pi Coding Agent. It has two parts:

- `/notes` — markdown summaries of Pi concepts and docs
- `/experiments` — small, self-contained code experiments, one folder per topic

## How to help

Fengwei is still learning Pi, so:

- Explain your reasoning as you go — don't just make changes silently
- Add comments in code that explain what each part does (why, not just what)
- When there are multiple approaches, briefly describe the trade-offs before choosing one
- Prefer simple and readable over clever

## Code style

Each experiment in `/experiments` should be self-contained in its own folder. Keep experiments small and focused on a single concept.
